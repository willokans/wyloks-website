# **Project Requirements Document: Wyloks Ltd App**

The following table outlines the detailed functional requirements of the Wyloks Ltd web application.

| Requirement ID | Description                        | User Story                                                                 | Expected Behavior/Outcome                                                                                                                                             |
|----------------|------------------------------------|----------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| FR001          | Delete existing homepage and create a new homepage | As a user, when I navigate to wyloks.com, I want to land on the homepage. | The app should show the homepage when the user navigates to the website `wyloks.com`. The homepage should include the following content and provide navigation to the About, Service, and Contact pages: <br><br>**Welcome to Wyloks Ltd**<br>We help businesses grow their online presence through strategic IT consultation, custom software development, and digital marketing.<br><br>**Our Core Services:**<br>- IT Consultation & Software Development<br>- SEO & Digital Marketing<br>- Affiliate Marketing (Amazon, Shopify & more)<br><br>**Why Choose Us?**<br>- End-to-end software development<br>- Proven SEO & marketing results<br>- Custom strategies for your business goals<br><br>**Explore Our Services** | **Get in Touch** |
| FR002          | Delete existing About page and create a new About page | As a user, when I navigate to wyloks.com, I want to land on the homepage and be able to navigate to the About page. | The app should display the About page when the user navigates to it. The About page should include navigation links to the Home, Service, and Contact pages. When the user clicks on **"See What We Offer"**, they should be redirected to the Service page. The About page should contain the following content:<br><br>**About Wyloks Ltd**<br>Wyloks Ltd is a registered UK company focused on helping businesses succeed online. We specialize in IT consultation, software development, and performance-driven marketing strategies.<br><br>**Our Mission**<br>To empower businesses of all sizes with modern digital tools and tailored growth strategies.<br><br>**Our Vision**<br>To become a trusted partner for businesses undergoing digital transformation across industries.<br><br>**Our Values**<br>- Integrity<br>- Innovation<br>- Transparency<br>- Results<br><br>**See What We Offer** |
| FR003          | Delete existing Service page if it exists and create a new Service page | As a user, when I navigate to wyloks.com, I want to land on the homepage and be able to navigate to the Service page. | The app should display the Service page when the user navigates to it. The Service page should include navigation to the Home, About, and Contact pages. When the user clicks **"Contact us today!"**, it should redirect them to the Contact page. The Service page should contain the following content:<br><br>**Our Services**<br><br>**1. IT Consultation & Software Development**<br>We provide end-to-end software solutions — from planning to deployment. Whether you're starting from scratch or need expert advice on your tech stack, Wyloks Ltd has you covered.<br><br>- Custom web & mobile applications<br>- Technical consulting & architecture<br>- Full-cycle development & support<br><br>**2. SEO & Digital Marketing**<br>We help businesses rank higher and attract the right customers through data-backed strategies.<br><br>- On-page & off-page SEO<br>- Content creation<br>- Social media marketing<br>- PPC and email campaigns<br><br>**3. Affiliate Marketing**<br>Monetize your content and drive revenue through affiliate programs with platforms like Amazon and Shopify.<br><br>- Affiliate setup & management<br>- Product research<br>- Performance optimization<br><br>**Ready to work with us? Contact us today!** |
| FR004          | Delete existing Contact or Contact Us page if it exists and create a new Contact page | As a user, when I navigate to wyloks.com, I want to land on the homepage and be able to navigate to the Contact page from any page. | The app should display the Contact page when the user navigates to it. The Contact page should include navigation links to the Home, About, and Service pages. If a user clicks on the **direct contact email**, the system should allow them to open their email client to send an email to `info@wyloksltd.com`. The page should also include a contact form where users can submit their **Name**, **Email**, and **Message**. This message should be sent to the admin's Gmail account without revealing the email address to the user. Upon successful submission, the system should display a toast or redirect to a confirmation page that informs the user: *"Your message has been sent. We will get back to you as soon as we can."*<br><br>The Contact page should contain the following content:<br><br>**Contact Wyloks Ltd**<br>Have a project in mind or need expert advice? We’d love to hear from you.<br><br>**Send us a message:**<br>- Name<br>- Email<br>- Message<br><br>**Direct Contact:**<br>Email: info@wyloksltd.com<br>Phone: +44 XXXX XXX XXX<br>Location: WYLOKS, N13 4BS, United Kingdom |
